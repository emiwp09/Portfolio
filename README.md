Este proyecto fue elaborado como parte de mi **servicio social**, en colaboración con mi compañero Martin Rule.

Desarrollar un **modelo predictivo** que permita determinar si los bebés nacerán **a término o pretérmino**, utilizando las variables disponibles en la base de datos.

## 📂 Contenido del repositorio

- Base de datos original (.xlsx)
